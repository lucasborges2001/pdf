# Docs (_pdf)

Este directorio es la **entrada única** para entender y modificar el framework.

- Empezar por: `docs/INDEX.md`
- Uso de CLI: `docs/USAGE.md`
- Especificación del formato `.txt`: `docs/FORMAT.md`

Reglas de mantenimiento:

1) Todo cambio de semántica (formato, CLI, layout, resolución de assets) se documenta en `docs/`.
2) Todo cambio con impacto en compatibilidad se refleja en `CHANGELOG.md`.
